Description

You are to construct a multiple view Android app for managing a collection of Place descriptions. Place properties were detailed in the first Lab Exercise. The app should be based on the first single view Android app that you developed. Add another class PlaceLibrary which initializes itself from a json file of places that are included in the app bundle. Your app should have a single view which lists the names of all of the Places in your library. Selecting one of the names transitions to a view which provides the details of the selected place. Your app should also include the following functionality:

  • Add a new place description to the library. The user manually enters all appropriate information.
  • Modify an existing place description.
  • Remove an existing entry from the library.
  • Distance and Initial Bearing. Provide the user the ability to query the Great Circle distance (statute miles) and initial bearing from one place to any other place in the library.
This app is similar to the first, but uses more complex controls and multiple views to display and manage a collection of PlaceDescription objects. Add a PlaceLibrary class which retains place description objects. This app should not need to use the network or a database to store entries in the PlaceLibrary. These will be added in future Labs. You are not required to persist any changes made to the PlaceLibrary. You should include a json file in the app bundle/project for initializing your PlaceLibrary. Create your own, or use the instructors: places.json .

Your app should allow the user to select one place name and see all of the information about that place, and it should provide the ability to add, change (all but the name), or remove place descriptions. New descriptions should be added to the collection by the user manually entering the appropriate information needed to create a PlaceDescription object. PlaceDescriptions should include at least the same properties as defined in the first Lab Exercise.

Note. Future assignments will utilize and build on this app by exploring local storage via a database, using the network to get data, synchronizing the two data storages approaches, and mapping.

Constraints

In the Android app adhere to the following constraints:

  1. Use an Action Bar that includes at least one button whose action relates to the requirements of this assignment.
  2. Use either a ListView or RecyclerView control with a properly formed adapter.
  3. Use at least the following basic controls: Button, EditText/TextView, and a Picker or Spinner.
  4. Provide separate classes for a place and for the place library.
